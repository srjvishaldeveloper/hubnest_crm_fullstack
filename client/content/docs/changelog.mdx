### Version History

#### v1.2.0 (June 2026)
- **Fumadocs Migration**: Migrated the legacy docs layout to Fumadocs for a premium, developer-friendly layout.
- **SaaS Theme Engine**: Implemented fully responsive theme components mapping light/dark variants natively.
- **HR & Support Modules**: Initial release of GPA Check-in, timesheet trackers, and SLA matrix configurations.
- **System Theme Auto-Sync**: Enabled automatic prefers-color-scheme client syncing inside `ThemeProvider`.

#### v1.1.0 (March 2026)
- **Twilio OTP Fallback**: AddedResend email OTP dispatcher to act as redundancy for SMS gateway timeouts.
- **Express connection pooling**: Integrated connection optimization layers inside Knex databases.
- **IndexedDB Syncing**: Integrated offline database synchronization structures for local offline lead queues.

#### v1.0.0 (January 2026)
- **Initial Release**: Basic CRM contact boards, Kanban pipelines, transaction ledgers, and authentication middleware.
